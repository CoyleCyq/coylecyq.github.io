获取URL的哈希

# 获取地址栏hash参数

# 方法名:

$.hash()

# 使用场景：

  • 获取url参数信息或者写入url

# 快速上手:

$.hash()  // 获取当前浏览器窗口的url信息
$.hash({text: '123'})
1
2

# 参数:

没有参数时是获取信息，设置时，参数是Object 注意:设置时，会覆盖掉之前url中参数， 而且只支持/FBA/FBADeliveryOrder#pageSize=50&orderStatus=2&searchtype=orderCode&keyword=111类似这种格式的url，此方法一般与$.queryToObject()连用

# 方法名:

$.queryToObject()

# 使用场景：

  • 将类似于"foo=bar&foo=baz&thinger=%20spaces%20=blah&zonk=blarg&"的字符串转成对象

# 快速上手:

$.queryToObject("foo=bar&foo=baz&thinger=%20spaces%20=blah&zonk=blarg&") 
// {
//		foo: [ "bar", "baz" ],
//		thinger: " spaces =blah",
//		zonk: "blarg"
// }

$.queryToObject($.hash())
